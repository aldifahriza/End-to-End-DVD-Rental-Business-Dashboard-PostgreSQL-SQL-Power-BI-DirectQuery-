# DVD Rental Business analytic

This repository contains an **end-to-end analytics project** for a DVD rental business. 
It showcases the full workflow from data extraction to visualization and reporting.

## Contents

- **SQL Scripts**: Queries used for extracting and aggregating rental and payment data from PostgreSQL.
- **Power BI Dashboard**: Interactive dashboard with DirectQuery connection to analyze revenue, rentals, and key metrics.
- **Presentation Slides (PPT)**: Summary of insights, methodology, and key findings for stakeholders.

## Key Features

- End-to-end workflow: From database queries to visual reporting.
- Monthly revenue and rental analysis.
- Insights on rental trends and revenue per transaction.
- Ready-to-use Power BI dashboard with live data connection.

## Technologies Used

- **PostgreSQL**: Database for storing rental and payment data.
- **SQL**: Queries for data extraction and transformation.
- **Power BI**: Dashboard creation and visualization.
- **DirectQuery**: Live data connection from Power BI to PostgreSQL.

## How to Use

1. Clone this repository.
2. Open the SQL scripts and adjust database connection settings if necessary.
3. Open the Power BI file and connect to your PostgreSQL database using DirectQuery.
4. Explore the interactive dashboard and analyze key metrics.
5. Refer to the PPT slides for a summary of insights and recommendations.
